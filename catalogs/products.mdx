---
title: 'Products & Variants'
description: 'Group items into products with variants'
---

Catalogs can group items (variants) under products.

## List products

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/products \
  -H "Authorization: Bearer rastro_pk_..."
```

```json
{
  "products": [
    {
      "product_id": "TSHIRT",
      "product_title": "Basic T-Shirt",
      "variants": [
        {"sku": "TSHIRT-RED-S", "color": "Red", "size": "S"},
        {"sku": "TSHIRT-RED-M", "color": "Red", "size": "M"}
      ]
    }
  ]
}
```

## Get product

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/products/{product_id} \
  -H "Authorization: Bearer rastro_pk_..."
```

## Update product fields

Update fields shared across all variants:

```bash
curl -X PUT https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/products/{product_id} \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "product_title": "Premium T-Shirt",
    "product_brand": "Apparel Co"
  }'
```

## Add variant

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/products/{product_id}/items \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "sku": "TSHIRT-BLUE-L",
    "color": "Blue",
    "size": "L"
  }'
```
