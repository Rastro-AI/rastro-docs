## Bulk upsert

Create or update up to 1000 items per request:

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items/bulk \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      {"sku": "A1", "title": "Product A", "price": 29.99},
      {"sku": "A2", "title": "Product B", "price": 39.99}
    ],
    "unique_field": "sku"
  }'
```

```json
{
  "created": 1,
  "updated": 1,
  "failed": 0,
  "total": 2
}
```

## Sync from database

```python
import requests

products = fetch_from_database()  # Your products

response = requests.post(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items/bulk",
    headers={
        "Authorization": "Bearer rastro_pk_...",
        "Content-Type": "application/json"
    },
    json={
        "items": products,
        "unique_field": "sku"
    }
)

print(response.json())
```

## Large datasets

For >1000 items, batch your requests:

```python
batch_size = 500

for i in range(0, len(products), batch_size):
    batch = products[i:i+batch_size]
    response = requests.post(url, headers=headers, json={
        "items": batch,
        "unique_field": "sku"
    })
    print(f"Batch {i//batch_size + 1}: {response.json()}")
```
