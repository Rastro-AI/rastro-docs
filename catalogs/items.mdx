---
title: 'GET List Items'
api: 'GET /public/catalogs/{catalog_id}/items'
---

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items \
  -H "Authorization: Bearer rastro_pk_..."
```

**Response:**

```json
{
  "items": [
    {"id": "item_123", "sku": "A1", "title": "Product A", "price": 29.99}
  ],
  "total": 150,
  "page": 1
}
```
