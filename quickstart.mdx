## 1. Get your API key

Go to [dashboard.rastro.ai/settings/api-keys](https://dashboard.rastro.ai/settings/api-keys) and create a key.

## 2. Enrich a product

```bash
curl -X POST https://catalogapi.rastro.ai/api/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "query": "iPhone 15 Pro 256GB",
    "fields": ["price", "currency"]
  }'
```

Response:

```json
{
  "query": "iPhone 15 Pro 256GB",
  "results": {
    "price": 999.00,
    "currency": "USD"
  },
  "sources": ["apple.com", "bestbuy.com"]
}
```

That's it. You just pulled live pricing data.
