title: 'Examples'
---

Quick examples to get started with Rastro.

## Enrich product data

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Find specifications and pricing"}'
```

## Run a flow

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/workflows/{flow_id}/execute \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"input": [{"sku": "ABC123", "name": "Widget"}]}'
```

## Bulk import items

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items/bulk \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"items": [{"sku": "A1", "title": "Product A"}], "unique_field": "sku"}'
```
