---
title: 'List Items'
api: 'GET /public/catalogs/{catalog_id}/items'
---

Get items from a catalog with pagination, search, and filtering.

## Request

```bash
curl "https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items?limit=50" \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Items to return (max: 1000)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Items to skip
</ParamField>

<ParamField query="search" type="string">
  Search text across all fields
</ParamField>

<ParamField query="sort" type="string">
  Sort by field, e.g., `price:asc`, `created_at:desc`
</ParamField>

<ParamField query="filter" type="string">
  Filter items, e.g., `price:>100`, `category:=Electronics`
</ParamField>


## Examples

### Basic listing

```python
response = requests.get(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items",
    headers={"Authorization": "Bearer rastro_pk_..."},
    params={"limit": 100}
)
items = response.json()["items"]
```

### Search

```bash
curl "https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items?search=laptop" \
  -H "Authorization: Bearer rastro_pk_..."
```

### Sort by price

```bash
curl "https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items?sort=price:asc" \
  -H "Authorization: Bearer rastro_pk_..."
```

### Filter expensive items

```bash
curl "https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items?filter=price:>500" \
  -H "Authorization: Bearer rastro_pk_..."
```

### Paginate through all items

```python
all_items = []
offset = 0
limit = 100

while True:
    response = requests.get(
        f"{BASE_URL}/public/catalogs/{catalog_id}/items",
        headers=headers,
        params={"limit": limit, "offset": offset}
    )
    data = response.json()
    all_items.extend(data["items"])

    if len(data["items"]) < limit:
        break
    offset += limit

print(f"Total items: {len(all_items)}")
```
