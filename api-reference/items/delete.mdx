---
title: 'Delete Item'
api: 'DELETE /public/catalogs/{catalog_id}/items/{item_id}'
---

Delete an item from a catalog.

## Request

```bash
curl -X DELETE https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items/item_xyz789 \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

<ParamField path="item_id" type="string" required>
  Item ID or unique identifier
</ParamField>

## Response

```json
{
  "message": "Item deleted successfully"
}
```

## Example

```python
response = requests.delete(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items/{item_id}",
    headers={"Authorization": "Bearer rastro_pk_..."}
)

if response.status_code == 200:
    print("Item deleted")
```
