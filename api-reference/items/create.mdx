---
title: 'Create Item'
api: 'POST /public/catalogs/{catalog_id}/items'
---

Create a single item. Uses upsert - updates if the item already exists.

## Request

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/items \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "unique_identifier": "LAPTOP-001",
    "data": {
      "title": "Pro Laptop",
      "price": 1299.99,
      "brand": "TechCorp"
    }
  }'
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

## Body

<ParamField body="unique_identifier" type="string" required>
  Value for the catalog's unique ID field
</ParamField>

<ParamField body="data" type="object" required>
  Item data (any fields)
</ParamField>

<ParamField body="source_info" type="object">
  Optional metadata about the source
</ParamField>

## Response

```json
{
  "id": "item_xyz789",
  "data": {
    "title": "Pro Laptop",
    "price": 1299.99,
    "brand": "TechCorp"
  },
  "metadata": {
    "created_at": "2024-01-15T10:30:00Z",
    "updated_at": "2024-01-15T10:30:00Z",
    "version": 1,
    "unique_identifier": "LAPTOP-001"
  }
}
```

## Examples

<CodeGroup>

```python Python
response = requests.post(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/items",
    headers={
        "Authorization": "Bearer rastro_pk_...",
        "Content-Type": "application/json"
    },
    json={
        "unique_identifier": "LAPTOP-001",
        "data": {
            "title": "Pro Laptop",
            "price": 1299.99,
            "brand": "TechCorp",
            "specs": {
                "cpu": "Intel i7",
                "ram": "16GB"
            }
        }
    }
)

item = response.json()
print(f"Created: {item['id']}")
```
