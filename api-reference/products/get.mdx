---
title: 'Get Product'
api: 'GET /public/catalogs/{catalog_id}/products/{product_id}'
---

Get a product with all its variants.

## Request

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/products/TSHIRT-BASIC \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

<ParamField path="product_id" type="string" required>
  Product ID
</ParamField>

{

  "product_title": "Basic T-Shirt",
  "product_brand": "Apparel Co",

    {
      "id": "item_123",

      "color": "Red",
      "size": "S",
      "price": 19.99,
      "in_stock": true
    },
    {
      "id": "item_124",
