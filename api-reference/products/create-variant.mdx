---
title: 'Create Variant'
api: 'POST /public/catalogs/{catalog_id}/products/{product_id}/items'
---

Add a new variant to an existing product.

## Request

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/products/TSHIRT-BASIC/items \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "sku": "TSHIRT-BASIC-BLUE-L",
    "color": "Blue",
    "size": "L",
    "price": 19.99
  }'
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

<ParamField path="product_id" type="string" required>
  Product ID
</ParamField>

## Body

<ParamField body="sku" type="string" required>
  Unique identifier for the variant
</ParamField>

Include any variant-specific fields:

```json
{
  "sku": "TSHIRT-BASIC-BLUE-L",
  "color": "Blue",
  "size": "L",
  "price": 19.99
}
```

## Response

```json
{
  "id": "item_125",
  "sku": "TSHIRT-BASIC-BLUE-L",
  "product_id": "TSHIRT-BASIC",
  "created_at": "2024-01-15T10:30:00Z",
  "message": "Variant created successfully"
}
```
