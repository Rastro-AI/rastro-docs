---
title: 'List Products'
api: 'GET /public/catalogs/{catalog_id}/products'
---

Get products with their variants grouped together.

## Request

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123/products \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Products to return
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Products to skip
</ParamField>

## Response

```json
{
  "products": [
    {
      "product_id": "TSHIRT-BASIC",
      "product_title": "Basic T-Shirt",
      "product_brand": "Apparel Co",
      "variants": [
        {
          "id": "item_123",
          "sku": "TSHIRT-BASIC-RED-S",
          "color": "Red",
          "size": "S",
          "price": 19.99
        },
        {
          "id": "item_124",
          "sku": "TSHIRT-BASIC-RED-M",
          "color": "Red",
          "size": "M",
          "price": 19.99
        }
      ]
    }
  ],
  "total": 150
}
```

## Example

```python
response = requests.get(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}/products",
    headers={"Authorization": "Bearer rastro_pk_..."}
)

products = response.json()["products"]
for product in products:
    print(f"{product['product_title']}: {len(product['variants'])} variants")
```
