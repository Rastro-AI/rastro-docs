---
title: 'Get Workflow Run Status'
api: 'GET /public/workflows/runs/{workflow_run_id}'
---

Check workflow execution status and get results.

## Request

```bash
curl https://catalogapi.rastro.ai/api/public/workflows/runs/run_xyz789 \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="workflow_run_id" type="string" required>
  Workflow run ID (from execute response)
</ParamField>

<ParamField query="page" type="integer" default="1">
  Results page number
</ParamField>

<ParamField query="page_size" type="integer" default="50">
  Results per page (max: 1000)
</ParamField>

## Response (Running)

```json
{
  "workflow_run_id": "run_xyz789",
  "status": "running",
  "progress": 0.6,
  "message": "Processing node 3 of 5",
  "results": null
}
```

## Response (Completed)

```json
{
  "workflow_run_id": "run_xyz789",
  "status": "completed",
  "progress": 1.0,
  "message": "Workflow completed successfully",
  "completed_at": "2024-01-15T10:35:00Z",
  "results": {
    "data": [
      {
        "sku": "LAPTOP-001",
        "title": "Pro Laptop",
        "ai_description": "High-performance laptop for professionals..."
      }
    ],
    "total": 2,
    "page": 1,
    "page_size": 50,
    "has_more": false
  }
}
```

## Status Values

| Status | Description |
|--------|-------------|
| `pending` | Queued |
| `running` | Processing |
| `completed` | Done |
| `failed` | Error occurred |
| `cancelled` | Stopped manually |

## Examples

### Poll with progress

```python
import time

while True:
    response = requests.get(
        f"https://catalogapi.rastro.ai/api/public/workflows/runs/{run_id}",
        headers={"Authorization": "Bearer rastro_pk_..."}
    )
    status = response.json()

    print(f"{status['progress']*100:.0f}% - {status['message']}")

    if status["status"] in ["completed", "failed", "cancelled"]:
        break

    time.sleep(3)

if status["status"] == "completed":
    results = status["results"]["data"]
    print(f"Got {len(results)} results")
```

### Paginate large results

```python
all_results = []
page = 1

while True:
    response = requests.get(
        f"{BASE_URL}/public/workflows/runs/{run_id}",
        headers=headers,
        params={"page": page, "page_size": 100}
    )
    data = response.json()

    if data["status"] != "completed":
        break

    all_results.extend(data["results"]["data"])

    if not data["results"]["has_more"]:
        break

    page += 1

print(f"Total results: {len(all_results)}")
```
