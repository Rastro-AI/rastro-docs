---
title: 'Get Catalog'
api: 'GET /public/catalogs/{catalog_id}'
---

Get a specific catalog by ID.

## Request

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123 \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

## Response

```json
{
  "id": "cat_abc123",
  "name": "Product Catalog",
  "description": "Main inventory",
  "unique_id_field": "sku",
  "item_count": 1523,
  "schema_version": "1.2",
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-15T10:30:00Z"
}
```

## Examples

<CodeGroup>

```python Python
response = requests.get(
    f"https://catalogapi.rastro.ai/api/public/catalogs/{catalog_id}",
    headers={"Authorization": "Bearer rastro_pk_..."}
)

catalog = response.json()
print(f"{catalog['name']}: {catalog['item_count']} items")
```

```javascript JavaScript
const response = await fetch(
  `https://catalogapi.rastro.ai/api/public/catalogs/${catalogId}`,
  { headers: { 'Authorization': 'Bearer rastro_pk_...' } }
);

const catalog = await response.json();
```

</CodeGroup>
