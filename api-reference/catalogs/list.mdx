---
title: 'List Catalogs'
api: 'GET /public/catalogs'
---

Get all catalogs in your organization.

## Request

```bash
curl https://catalogapi.rastro.ai/api/public/catalogs \
  -H "Authorization: Bearer rastro_pk_..."
```

## Parameters

<ParamField query="limit" type="integer" default="10">
  Number of catalogs to return (1-100)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of catalogs to skip
</ParamField>

## Response

```json
{
  "catalogs": [
    {
      "id": "cat_abc123",
      "name": "Product Catalog",
      "description": "Main inventory",
      "unique_id_field": "sku",
      "item_count": 1523,
      "created_at": "2024-01-01T00:00:00Z"
    }
  ],
  "total_count": 1,
  "page": 1,
  "page_size": 10
}
```

## Examples

<CodeGroup>

```python Python
import requests

response = requests.get(
    "https://catalogapi.rastro.ai/api/public/catalogs",
    headers={"Authorization": "Bearer rastro_pk_..."},
    params={"limit": 10}
)

catalogs = response.json()["catalogs"]
for cat in catalogs:
    print(f"{cat['name']}: {cat['item_count']} items")
```

```javascript JavaScript
const response = await fetch(
  'https://catalogapi.rastro.ai/api/public/catalogs?limit=10',
  { headers: { 'Authorization': 'Bearer rastro_pk_...' } }
);

const { catalogs } = await response.json();
catalogs.forEach(cat => console.log(`${cat.name}: ${cat.item_count} items`));
```
