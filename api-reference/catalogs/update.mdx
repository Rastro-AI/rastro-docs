---
title: 'Update Catalog'
api: 'PUT /public/catalogs/{catalog_id}'
---

Update catalog name or description.

## Request

```bash
curl -X PUT https://catalogapi.rastro.ai/api/public/catalogs/cat_abc123 \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"name": "Updated Name"}'
```

## Parameters

<ParamField path="catalog_id" type="string" required>
  Catalog ID
</ParamField>

## Body

<ParamField body="name" type="string">
  New catalog name
</ParamField>

<ParamField body="description" type="string">
  New description
</ParamField>

## Response

```json
{
  "id": "cat_abc123",
  "name": "Updated Name",
  "description": "Main inventory",
  "unique_id_field": "sku",
  "updated_at": "2024-01-15T10:30:00Z"
}
```
