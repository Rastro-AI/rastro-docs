---
title: 'Create Catalog'
api: 'POST /public/catalogs'
---

Create a new catalog.

## Request

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/catalogs \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Catalog",
    "unique_id_field": "sku",
    "description": "Product inventory"
  }'
```

## Body

<ParamField body="name" type="string" required>
  Catalog name
</ParamField>

<ParamField body="unique_id_field" type="string" required>
  Field used to identify items uniquely (e.g., `sku`, `product_id`)
</ParamField>

<ParamField body="description" type="string">
  Optional description
</ParamField>

<ParamField body="product_enabled" type="boolean" default="false">
  Enable product/variant grouping
</ParamField>

## Response

```json
{
  "id": "cat_abc123",
  "name": "My Catalog",
  "description": "Product inventory",
  "unique_id_field": "sku",
  "item_count": 0,
  "created_at": "2024-01-15T10:30:00Z"
}
```

## Examples

<CodeGroup>

```python Python
import requests

response = requests.post(
    "https://catalogapi.rastro.ai/api/public/catalogs",
    headers={
        "Authorization": "Bearer rastro_pk_...",
        "Content-Type": "application/json"
    },
    json={
        "name": "Electronics",
        "unique_id_field": "sku",
        "description": "Electronic products"
    }
)

catalog = response.json()
print(f"Created catalog: {catalog['id']}")
```

```javascript JavaScript
const response = await fetch('https://catalogapi.rastro.ai/api/public/catalogs', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer rastro_pk_...',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Electronics',
    unique_id_field: 'sku',
    description: 'Electronic products'
  })
});

const catalog = await response.json();
console.log(`Created catalog: ${catalog.id}`);
```
