**Flows** are data pipelines you build in the [Rastro dashboard](https://dashboard.rastro.ai), then execute via API.

## How it works

1. **Build in the dashboard** — Create your flow visually at [dashboard.rastro.ai](https://dashboard.rastro.ai)
2. **Get your Flow ID** — Copy it from the URL: `dashboard.rastro.ai/flows/{FLOW_ID}/edit`
3. **Execute via API** — Send data in, get enriched data out

## Example

```bash
# Execute
curl -X POST https://catalogapi.rastro.ai/api/public/workflows/{FLOW_ID}/execute \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "input": [
      {"sku": "A1", "title": "Product A"},
      {"sku": "A2", "title": "Product B"}
    ]
  }'

# Response
{
  "workflow_run_id": "run_abc123",
  "status": "running"
}
```

```bash
# Check status
curl https://catalogapi.rastro.ai/api/public/workflows/runs/run_abc123 \
  -H "Authorization: Bearer rastro_pk_..."

# Response (when complete)
{
  "status": "completed",
  "results": {
    "data": [
      {"sku": "A1", "title": "Product A", "enriched_field": "..."},
      {"sku": "A2", "title": "Product B", "enriched_field": "..."}
    ]
  }
}
```
