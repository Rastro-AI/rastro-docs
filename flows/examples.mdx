---
title: 'Examples'
---

Use Flows when you need custom data processing pipelines. Flows are built visually in the [Rastro dashboard](https://dashboard.rastro.ai), then executed via API.

## Execute a flow

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/workflows/{flow_id}/execute \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"input": [{"sku": "ABC", "title": "My Product"}]}'
```

Response:
```json
{
  "run_id": "run_xyz",
  "status": "running"
}
```

## Check flow status

```bash
curl https://catalogapi.rastro.ai/api/public/workflows/runs/{run_id} \
  -H "Authorization: Bearer rastro_pk_..."
```

Response:
```json
{
  "status": "completed",
  "results": {
    "data": [{"sku": "ABC", "title": "My Product", "enriched_field": "..."}]
  }
}
```

## Wait for completion

Add `wait_for_result=true` to block until done (max 5 min):

```bash
curl -X POST "https://catalogapi.rastro.ai/api/public/workflows/{flow_id}/execute?wait_for_result=true" \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{"input": [{"sku": "ABC"}]}'
```
