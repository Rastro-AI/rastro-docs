## Get your API key

1. Go to [dashboard.rastro.ai/settings/api-keys](https://dashboard.rastro.ai/settings/api-keys)
2. Click **Create New Key**
3. Copy the key (it won't be shown again)

## Use your API key

Include it in the `Authorization` header:

```bash
curl https://catalogapi.rastro.ai/api/enrich \
  -H "Authorization: Bearer rastro_pk_your_key_here"
```

## Key format

```
rastro_pk_live_abc123...   # Production key
rastro_pk_test_abc123...   # Test key
```

## Errors

| Code | Meaning |
|------|---------|
| `401` | Missing or invalid API key |
| `403` | Key doesn't have access to this resource |
