## Installation

```bash
pip install rastro
```

## Quick Start

```python
from rastro import Rastro

client = Rastro(api_key="rastro_pk_...")

# List catalogs
catalogs = client.catalogs.list()

# Get items
items = client.catalogs.items.list(catalog_id="cat_abc123", limit=100)

# Bulk upsert
result = client.catalogs.items.bulk_upsert(
    catalog_id="cat_abc123",
    items=[
        {"sku": "A001", "title": "Product A", "price": 29.99},
        {"sku": "A002", "title": "Product B", "price": 39.99},
    ],
    unique_field="sku"
)
print(f"Created: {result.created}, Updated: {result.updated}")
```

## Workflows

```python
# Execute workflow and wait for results
results = client.workflows.execute_and_wait(
    workflow_id="wf_abc123",
    input=[
        {"sku": "A001", "title": "Product A"}
    ]
)

for item in results:
    print(item)
```

## Async Support

```python
import asyncio
from rastro import AsyncRastro

async def main():
    client = AsyncRastro(api_key="rastro_pk_...")

    catalogs = await client.catalogs.list()
    print(catalogs)

asyncio.run(main())
```

## Error Handling

```python
from rastro import Rastro, RastroError, AuthenticationError

client = Rastro(api_key="rastro_pk_...")

try:
    catalog = client.catalogs.get("invalid_id")
except AuthenticationError:
    print("Invalid API key")
except RastroError as e:
    print(f"API error: {e.message}")
```
