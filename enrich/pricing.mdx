## Get current price

```bash
curl -X POST https://catalogapi.rastro.ai/api/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "query": "AirPods Pro 2nd generation",
    "fields": ["price", "currency", "availability"]
  }'
```

```json
{
  "query": "AirPods Pro 2nd generation",
  "results": {
    "price": 249.00,
    "currency": "USD",
    "availability": "in_stock"
  },
  "sources": ["apple.com"]
}
```

## Batch pricing

```bash
curl -X POST https://catalogapi.rastro.ai/api/enrich/batch \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "items": [
      {"sku": "A1", "query": "iPhone 15 Pro"},
      {"sku": "A2", "query": "iPhone 15"},
      {"sku": "A3", "query": "iPhone SE"}
    ],
    "fields": ["price", "currency"]
  }'
```
