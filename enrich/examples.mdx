---
title: 'Examples'
---

Use Enrich when you want Rastro to research items on the web and fill in missing data. Rastro scrapes manufacturer sites, extracts specs, and tracks sources for every field.

## Single item

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Find product specifications",
    "items": [{"name": "Sony WH-1000XM5"}]
  }'
```

## Multiple items

Items are processed concurrently:

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Find pricing and availability",
    "items": [
      {"name": "iPhone 15 Pro"},
      {"name": "Samsung Galaxy S24"},
      {"name": "Google Pixel 8"}
    ]
  }'
```

## Custom fields

Pass any data you have, Rastro will use it for better research:

```bash
curl -X POST https://catalogapi.rastro.ai/api/public/enrich \
  -H "Authorization: Bearer rastro_pk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Find technical specifications and reviews",
    "items": [
      {"name": "MacBook Pro 14", "brand": "Apple", "category": "Laptops"}
    ]
  }'
```
